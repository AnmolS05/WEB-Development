# To-Do List App ✅

A simple web app to add, complete, and delete daily tasks.

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript (Vanilla)

## ✨ Features
- Add tasks
- Mark tasks as done
- Delete tasks
- Instant UI updates (no page reload)

## 📂 Files
- `index.html` – app structure  
- `style.css` – basic styling  
- `script.js` – handles all interactions
